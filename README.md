🍕 Pizza Sales Report | Power BI Dashboard
Project Overview

This project analyzes pizza sales performance using Power BI to provide insights into:

Revenue performance
Order trends
Product category performance
Pizza size contribution
Daily and monthly sales patterns

The dashboard helps stakeholders identify top-performing products, customer preferences, and sales trends throughout the year.

Business Questions

The dashboard answers the following questions:

What is the total revenue generated?
How many orders were placed?
How many pizzas were sold?
Which pizza category performs best?
Which pizza size generates the highest sales?
What are the busiest days of the week?
How do sales fluctuate throughout the year?
Data Preparation

The dataset was cleaned and transformed using Power Query.

Data Cleaning Steps
Removed duplicate records
Checked and handled missing values
Standardized data types
Created a Date Table
Built relationships between tables
Added calculated columns and measures
Data Modeling

A star schema model was implemented:

Fact Table
Orders
Order Details
Dimension Tables
Date
Pizza Category
Pizza Size

Relationships were created using primary and foreign keys to enable accurate filtering and analysis.

KPIs
Total Revenue
Total Pizzas Sold
Total Orders
Average Orders
